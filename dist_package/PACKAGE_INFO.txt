CETAMURA BATCH INGEST TOOL - DISTRIBUTION NOTES
================================================

Version: 1.3.0
Date: April 20, 2026

This directory keeps distribution helper files and user-facing package
documentation. Release packages should be generated from the repository root
with:

  .\scripts\build\create_dist_package.ps1 -Version 1.3.0

The generated package includes:

  src/
    main.py
    validation.py

  requirements/
    requirements.txt

  docs/
    readme.md
    RELEASE_CHECKLIST.md

  scripts/build/
    build_exe.ps1
    build_exe_macos.sh
    build_cross_platform.sh

  VERSION_INFO.txt

If a Windows executable already exists at dist/CetamuraBatchIngest.exe when the
package script is run, it is copied into an executables/ folder in the generated
package.

Runtime requirements:

  - Windows 10 or 11 for the primary supported desktop workflow
  - Python 3.9+
  - Pillow
  - PyMuPDF
  - defusedxml

Primary release checks:

  python -m pytest tests/ -v --tb=short
  python tests/run_tests.py
  python -m black --check ...
  python -m flake8 --jobs 1 ...
  python -m bandit -r src -ll
